📖How To Use Github📖
  • 1. 깃 설치
    이 Git 바로가기 링크로 들어간후 Downloads 클릭
  • 각자 OS에 맞게 선택후 설치
  • 깃 설치후 터미널 창에서 git version
    입력 후 위와 같이 버전이 나온다면 설치 성공!
  • 2. 깃허브 계정 만들기
    이 GitHub 바로가기 링크로 들어간후 Sign up 클릭
  • 계정을 생성 해줍니다
  • 일단 Skip this for now 클릭!
  • 이 창 까지 오시면 성공 한 것 입니다!
  • 2. 깃허브 repository 생성 및 파일 업로드
    create repository 클릭!
  • Repository name 설정후 바로 Create repository 클릭!
  • 이 창 까지 오시면 잠시 멈추시고
  • 아무 폴더나 생성 후 생성한 파일에 들어갑니다
  • 파일 경로창 클릭후 cmd 입력을 하면
    해당 경로를 시작으로 하는 터미널이 나타납니다.
  • 열린 터미널 창에 git init 입력 후 엔터!
  • 폴더에 아무 파일이나 생성!
  • git add * 입력 후 엔터
  • git commit -m "원하는 메시지 입력" 입력 후 엔터
    처음 하신 분 들은 위와 같이 나오시는게 정상입니다.
  • git config --global user.email "깃허브 가입한 이메일" 입력 후 엔터
    git config --global user.name "깃허브 이름" 입력 후 엔터
    다시 git commit -m "원하는 메시지 입력" 입력 후 엔터
    위와 같이 나오면 성공 한 것입니다.
  • repository 로 돌아간 후 위 사진과 동일한 라인의 명령어 복사
    git remote add origin https://github.com/"자신의_이름"/"repository이름".git
  • 복사한 명령어 터미널에 붙여넣기 후 엔터
    git remote add origin https://github.com/"자신의_이름"/"repository이름".git
  • 터미널에 git push origin master 입력 후 엔터
    처음 하시면 깃허브의 이메일(Username)과 password를 입력하라고 뜹니다.
    입력후 엔터를 치시면 됩니다. 위와 같은 결과이면 성공!
  • 이제 만들어 두었던 repository에 들어가면
    저희가 폴더에 만들었던 파일들이 repository에 올라가 있을 것 입니다!
🎉축하합니다!🎉
기본적인 깃과 깃 허브의 사용법을 배우셨습니다!
